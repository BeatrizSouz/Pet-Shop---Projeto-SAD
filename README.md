# Pet-Shop — Projeto Sistemas de Apoio à Decisão

## 📌 Descrição
O projeto tem como finalidade aplicar os conceitos de **Sistemas de Apoio à Decisão (SAD)** em um ambiente de Pet Shop, utilizando-o como **caso de estudo** para consolidar os conhecimentos adquiridos durante o período letivo.
